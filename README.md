
# Análisis de Tiendas – Alura Store

Este proyecto analiza los datos de ventas de las cuatro tiendas de la cadena **Alura Store** con el objetivo de identificar cuál de ellas debería venderse para iniciar un nuevo emprendimiento.

## Objetivo
Evaluar el desempeño de cada tienda a partir de:
- Ingresos totales
- Categorías más relevantes
- Calificación promedio de clientes
- Productos más y menos vendidos
- Costos de envío

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Conclusión
A partir del análisis realizado, se recomienda vender la **Tienda 4**, ya que presenta los menores ingresos, una menor satisfacción promedio de los clientes y un menor desempeño general frente a las demás tiendas.
